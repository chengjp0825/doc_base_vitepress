# Bosch BME280

BME280 是博世 Sensortec 推出的一款集成温度、湿度和气压测量的数字传感器。由于其极小的封装和低功耗，是气象站和高度计的理想选择。

## 技术规格
- **接口**: I2C (最大 3.4 MHz) 和 SPI (3线和4线，最大 10 MHz)
- **电压**: 1.71 V to 3.6 V
- **电流消耗**: 
  - 1.8 µA @ 1 Hz 测量频率（仅读取温湿度）
  - 3.6 µA @ 1 Hz 测量频率（温湿度气压全部开启）

## I2C 地址
BME280 的 I2C 地址可通过 `SDO` 引脚设置：
- `SDO` 接 GND: `0x76`
- `SDO` 接 VDDIO: `0x77`

::: tip 数据补偿
传感器的原始读数必须经过芯片内部的校准工厂参数进行补偿计算。博世官方提供了现成的 C 驱动库来进行庞大的浮点数或定点数补偿计算。
:::
