# ESP32 硬件设计指南

ESP32 是乐鑫科技 (Espressif) 推出的一款集成了 2.4 GHz Wi-Fi 和双模蓝牙功能的单芯片，采用台积电超低功耗 40 nm 工艺。

## 系统框图

ESP32 包含两个可以单独控制的 Xtensa® 32-bit LX6 微处理器，时钟频率可调范围为 80 MHz 到 240 MHz。

## 典型的外围电路

要让 ESP32 正常启动，硬件设计上需要注意以下几个 Strap 引脚的状态：

1. **GPIO0**: 决定启动模式。低电平进入下载模式，高电平（默认）进入 Flash 运行模式。
2. **GPIO2**: 启动时必须为悬空或低电平。
3. **GPIO12**: 内部 LDO 电压选择。如果接高电平，内部 Flash 将无法正常工作。

::: warning 注意
ESP32 的瞬态电流可以在射频发射时达到 500mA 以上，因此所配 LDO (如 AMS1117-3.3) 必须保证足够的供电能力和良好的去耦回路。
:::
