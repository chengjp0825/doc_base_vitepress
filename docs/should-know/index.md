# 你应该知道的

这是一个通识专题，内容包括但不限于你应该知道的，你可能知道的，你或许想要知道的。但是笔者认为，以上这些类型的内容，你都**应该知道**。

它不追求把每个话题讲成厚重手册，而是先帮你建立稳定的工程判断框架：

- 什么概念必须先搞懂。
- 什么问题在项目里最常出现。
- 什么做法能在真实场景里减少踩坑。

同样的，如果在阅读时发现了任何问题，或者有更好的改进建议，随时欢迎提 Issue 或 PR 来共同完善！

## 本章定位

你可以把这一章理解为“工程常识速查与方法论入口”。

- 对新手：用于快速建立全局认知，减少只学局部知识带来的断层。
- 对进阶读者：用于补齐跨主题细节，提升方案评审与问题定位效率。
- 对团队协作：用于统一讨论语境，减少沟通成本。
