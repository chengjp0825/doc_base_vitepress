# 差分信号

很多人第一次接触差分信号（Differential Signaling）时，会把它理解成“把一根线变成两根线”。这个理解不算错，但远远不够。

差分真正改变的，不仅仅是导线数量，更是判决方式：接收端从“看那唯一传输线上的信号波形对地电压”，改成“看两根传输线信号波形之间的电压差”。

本文将聚焦“差分”二字的物理内涵。。低压差分信号（Low-Voltage Differential Signaling，LVDS）只在后文点到为止，作为经典例子引入。至于 LVDS 的参数细节和设计实践，会在后续有专文展开。

---

## 1. 为什么要学差分信号

以下现象在实际场景下并不少见。

一条传输线在低速下跑得很稳，速率一提上去，偶发误码就出现了，驱动端输出1010 1010，接收端却接收0000 0000。协议没变，逻辑没变，代码也没变，结果却变差了。

这类问题继续在代码层面排查注定是徒劳的，因为它的根本原因通常不是数字逻辑，而是电气判决环境变差了。

单端信号（Single-Ended Signaling）在高速信号传输场景下常见有如下三个压力：

- 高度的参考平面依赖与地弹效应: 接收判决依赖信号对地电压是否越过阈值（$V_{th}$），地弹噪声（Ground Bounce）会直接影响判决裕量。
::: warning
不过需要注意，虽然差分信号通过互补参考缓解了地弹问题，但其极度惧怕物理不对称性，任何走线偏斜都会导致优势丧失并激发共模噪声。
:::

- 不可控的回路面积与电磁干扰: 单端信号的回流路径通常较难控制，导致等效回路面积偏大。随着信号边沿速率的不断提升（高频谐波剧增），过大的回路不仅会成倍放大对外的高频辐射，也使其更容易感应并吸收外界的电磁干扰。

- 阻抗敏感与串扰导致的眼图压缩: 在复杂的物理链路中，单端信号对阻抗不连续极度敏感。寄生参数引起的信号反射，叠加相邻网络间的耦合串扰，会导致系统的等效判决窗口迅速缩小，进而引发眼图严重闭合。

高速传输设计的核心早已不再是单纯追求速率，而是比拼极限条件下的信号完整性与抗干扰能力。这正是差分信号能够克服单端信号瓶颈，成为现代高速通信主流架构的深层次原因。

## 2. 差分信号的物理机制

![](http://picbed.moongazer.cn/hub/diffsignal_white.png)

差分对（P 线与 N 线）在物理实体上表现为两根阻抗受控的传输线，其本质是接收端“差分采样器”的前置传输介质。驱动器会在 P 线与 N 线上同时发送摆幅相等、相位互补的两个电信号 $V_{p}$ 与 $V_{n}$。

由于接收端是通过计算两线之间的相对差值来确定判决阈值（$V_{th}$），而非直接测量信号相对于物理参考地的绝对电位，因此差分信号的逻辑区间由以下物理量定义：

- 差模电压（Differential Mode Voltage）：决定信号的逻辑极性与有效幅值。
$$
V_{diff} = V_{p} - V_{n}
$$

- 共模电压（Common Mode Voltage）：表示两线的直流偏置或同步波动量。
$$
V_{cm} = \frac{V_{p} + V_{n}}{2}
$$

接收端的比较器仅对差模电压 $V_{diff}$ 响应。当外部电磁干扰（EMI）等量耦合至 P 线与 N 线时，该同向扰动在系统内表现为共模电压的波动（$\Delta V_{cm}$），也称共模噪声 $V_{Noise}$ 。由于差分采样网络的判决逻辑不依赖于系统参考地，该共模扰动会在接收端的相减操作中被物理隔离。
$$
V'_{diff} = (V_{p} + V_{Noise}) - (V_{n} + V_{Noise}) = V_{p} - V_{n} = V_{diff}
$$

这种对共模扰动的隔离程度由接收器的 **共模抑制比（Common-Mode Rejection Ratio, CMRR）** 决定，记作 $A_{cm}$。共模噪声在系统内部残留的等效误差近似满足：

$$
V_{err,cm} \approx \frac{V_{cm,noise}}{10^{A_{cm}/20}}
$$

以典型参数为例：当 $A_{cm} = 30\text{ dB}$ 时，物理链路上高达 $100\text{ mV}$ 的共模噪声，在经过接收端网络后，仅产生约 $3.16\text{ mV}$ 的等效电压误差。

::: warning ❓ 那我问你：差分信号的两根走线靠得那么近，为什么它们之间不会产生破坏性的串扰（Crosstalk）？
面试官冷笑一声：“大家都知道布线要遵循 3W 原则来防串扰。既然你两根线贴得这么近，难道在差分信号这里，物理学定律就失效了吗？”

::: details 💡 点击查看满分回答
**差分线内部的相互作用不属于非预期的“串扰”，而是设计上预期的“电磁耦合（Coupling）”。**

1. **磁场相消与能量约束**：差分信号的P线和N线传输的是大小相等、方向相反的电流。根据右手螺旋定则，两者产生的磁场方向完全相反。当两根线紧密靠近时，它们在导线外部的磁场会相互抵消（不对外产生辐射和串扰）；而在导线内部，磁场紧密交织，将电磁能量牢牢“束缚”在两根线之间，防止能量外泄。
2. **预期的耦合构成“差分阻抗”**：串扰的本质是非预期的、随机的能量耦合。而差分对中P与N之间的电磁感应是预期的、同步的。在设计时，已经将这种强烈的相互耦合计算在内。这种耦合直接构成了传输线的“差分阻抗”（如标准的 100Ω）。预期耦合构成差分行为，但是否反射仍取决于阻抗连续性与端接条件。
3. **紧密耦合实现“共模抑制（CMRR）”**:差分走线靠得越紧，意味着外部干扰源到达P线和N线的空间距离越趋于绝对相等。距离相等使得外部干扰在两根线上感应出的噪声幅度相同、相位相同，这被称为“共模噪声”。

**总结**：差分线之间不是不可控的串扰，而是精密设计的耦合。它们通过紧密靠拢，对外实现了磁场抵消与共模免疫，对内实现了能量锁定与阻抗稳定。对差分信号而言，“距离产生美”是错的，“亲密无间”才是维持高速稳定的核心密码。

## 3. 差分信号的优势与代价

差分信号并不是绝对的好，它本质上是性能与成本、空间之间的一种权衡。

### 3.1 主要优势

- 辐射更容易控制   
差分对中的两根导线不仅紧密耦合，而且始终传输着大小相等、方向相反的电流。根据电磁学原理，这两股反向电流产生的磁场极性相反，在对称、回流完整、端接合理时，在空间中会相互纠缠并完美抵消。这使得差分信号即使在极高的频率下运行，对外界的电磁辐射也极小。此处会在5.3节展开说明。


- 抗共模干扰能力强  
这是差分信号最本质的优势。在实际传输中，外界的电磁干扰或环境噪声通常会同等、均匀地耦合到两根差分线上，形成共模噪声，而接收端仅通过计算两者的电压差来判决逻辑，这些相同的噪声会被数学相减并完全抵消。

![](http://picbed.moongazer.cn/hub/20260416164159675.png)
(图片来源：wiki百科)

- 地电位偏移的容忍度高   
单端信号严重依赖于系统地（Ground）作为参考，而差分信号是相互参考的。因此，即使发送端和接收端之间存在地电位差或同步开关噪声（SSO），也不会影响差分信号的逻辑判定

- 适合更高边沿速率和更紧张的时序预算   
差分判决不依赖绝对电压阈值，差分信号可以采用极低的电压摆幅（如LVDS、CML），从而实现信号在逻辑高低之间切换所需的时间更短（边沿速率更快）。精确的交叉点特性也显著降低了时序抖动（Jitter），轻松应对高速串行总线（Gbps级别）极其紧张的时序预算。

下一章将围绕前两点进行更深层次原因探讨。

面试官冷笑一声：“你都说差分抗干扰了，那为什么最后还是会丢包、误码，甚至链路起不来？”

**会发生的不是单点退化，而是整条链路的判决裕量被系统性吞噬。**

1. **等长失配导致时序偏移（Skew）**：两根线传播延时不一致时，接收端在同一采样时刻看到的 $V_p$ 与 $V_n$ 不是同一比特“瞬间值”，差模幅度被压缩，眼图横向收窄，时序裕量下降。
2. **等距与对称性破坏导致模态转换**：线间耦合不稳定会把本该在差模通道里的能量转换成共模分量，抗共模干扰优势被削弱，对外辐射和对内噪声敏感度同时上升。
3. **阻抗不连续导致反射叠加**：过孔、线宽突变、参考平面不连续或端接不匹配会引入反射，反射与主信号叠加后造成过冲、振铃和眼图闭合，最终表现为误码率上升。

**总结**：差分信号并不自动“免疫问题”，它只是把系统可靠性建立在更严格的几何对称与阻抗连续之上。版图细节做不好，优势会快速反转为风险。

**如果线长已经很接近了，是否就可以忽略阻抗控制和端接？**

:::

## 4. 差分对的三板斧

从PCB物理和电磁学的角度来看，差分信号传输线之所以具备极低的电磁辐射（EMI）和极强的抗干扰能力，核心在于
**“等大反相”的电磁场抵消效应**以及**接收端的“共模抑制”机制**。本章对较为抽象的“电磁场抵消效应”展开。

下图为**差分线的奇模与偶模传播模式对比**图。

（图4.1     图片来源：S. H. Hall et al., High-Speed Digital Designs, Chap.7）   

图4.1 左半图为**奇模传播 (Odd-mode Propagation)**，差分对两根走线上的电流大小相等、方向、电位相反，大量大量电场线（E-field）集中在两根走线之间，形成紧密的互容耦合。由于电流方向相反，两线产生的磁场（H-field）在外部区域发生抵消，能量被有效“束缚”在走线对内部。详见图4.2。

（图4.2     图片来源：S. H. Hall et al., High-Speed Digital Designs, Chap.7）

## 5. 差分设计重点

前文已完成差模/共模、传播模式与权衡关系的说明；本节聚焦阻抗连续性、对称性与回流路径，其中回流路径涉及SI/PI，在PCB Layout部分涉及较多，故在此处展开会占据较多篇幅。

### 5.1 阻抗连续性
链路中的过孔、连接器、焊盘突变都会造成阻抗突变，进而带来反射。反射不一定马上导致通信失败，但会稳定地吞掉眼图余量。在实际布线中，差分线的阻抗受线宽、线距、铜厚、介质厚度等参数严格制约。设计时不仅要保持这些物理参数的稳定，还要兼顾工厂的实际制程能力。例如，盲目追求高阻抗而把线宽设得过窄，会导致蚀刻偏细甚至断裂，反而使阻抗超标。

### 5.2 对称性与等长
差分对两根线如果传播延时不一致，差模能量会被转换为共模分量。换句话说，你本来想用差分抵抗共模，结果自己制造了共模。因此，差分走线设计中最核心的规则就是匹配线长。除了长度一致，两根线的线宽、线距以及到参考层的距离也必须完全对称，否则就会出现阻抗失衡。如果必须使用蛇形走线进行等长补偿，拐角应使用45°或圆弧，并保持线间距大于等于3倍线宽以减少串扰。

### 5.3 参考平面完整性（重点）

即使在设计阶段到Layout阶段，都小心小心再小心以上两点，但是很多问题不是发生在“差分线本身”，而是发生在“回流路径被破坏”。一旦跨分割平面，链路稳定性和辐射表现通常会同时变差。设计中存在一个常见误区，即认为**差分线能彼此互为回流路径**。

#### **理想模型和现实耦合的差异**

理想情况下，差分对传输着大小完全相等、方向完全相反的电流（$+i$ 与 $-i$），因为这两个电流的大小相等且符号相反，它们的数学总和（$+i$ + $-i$）正好为0。此时差分对被认为是平衡差分对。此时，其中一根走线完美地充当了另一根走线的回流路径，确实不需要系统参考平面参与。具体表现为：

- 两根走线正中间形成“虚拟参考平面”

- 没有任何电流会流经系统参考平面

**然而，在现实的PCB设计中，差分对的净电流几乎不可能为0，这种差分对被称为不平衡差分对。**   
在实际电路的不平衡差分对中，**源端芯片的电气非对称**和**PCB Layout的物理非对称**是造成此现象的主要原因。前者属于先天失调，主要表现为：相位偏差 (Time Skew)、边沿速率不匹配 (Rise/Fall Time Mismatch)以及幅度不平衡 (Amplitude Mismatch)。；后者属于后天畸变，主要表现为5.1节与5.2节中强调的内容。两个信号在转换时无法保持绝对的同步与对称，将导致 $+i$ 和 $−i$ 不完全相等时，它们的总和 ($+i$ + $−i$ ) 就不再是0。这个不为 0 的净电流（通常表现为共模电流）就是被打破平衡后多出来的“游离”电流。那这“游离电流”该何去何从？ 

#### **最小电感路径原则**

电流必须在一个闭合的环路中流动，**且会选择最小阻抗回路**，这是所有电气信号的基本法则。这部分多出来的“游离”电流已经无法通过另一根处于饱和或相位错位的差分线进行回流，为了完成闭合回路，这部分"游离"电流必须被强制寻找其他的物理路径返回源端。理论上，差分电流存在两条潜在路径：
- 相邻的差分线（排除）
- 系统参考平面   

“它不去地平面还能去哪？”。   

此处强调的系统参考平面并不一定是地平面，且上文同样强调“选择最小阻抗回路”，若在叠层设计中，紧贴差分对的是一个电源平面（Power Plane），那么这个电源平面也会因为“最小阻抗”而充当它的参考回路。   
::: warning
若使用电源平面作为交流回流路径，必须确保在信号源端和接收端附近，有充足的去耦电容将电源平面与真实的系统地平面进行低阻抗的高频连接，否则会引发严重的 EMI 问题。
:::
在分析参考平面上的回流电流时，主要关注回路的串联阻抗，串联阻抗（$Z$）主要由电阻（$R$）和感抗（$j \omega L$）组成。在低频状态下，感抗很小，信号回流会遵循“电阻最小”的原则，选择物理距离最短的直线路径返回源端；所下图示的直流电流的流动方式，选取了接地层中从过孔1至过孔2的电阻最小的路径。虽然会发生一些电流扩散，但基本上不会有电流实质性偏离这条路径。
![](http://picbed.moongazer.cn/hub/20260417112306745.png)
（图片来源：Hank Zumbahlen. staying-well-grounded）   

随着信号频率升高 ~~（通常>10MHz）~~（进入 MHz 频段），感抗急剧增加，成为阻抗组成部分中的绝对主导因素，此时电流仍会寻找“阻抗最小的路径”，更准确地说，电流会流经“电感最小”的路径。

**电感大小取决于“电流环路面积”**   

在电磁学中，决定一段路径电感大小的核心物理量是电流的环路面积（Loop Area）。信号从源端出发再返回的路径在空间围成了一个闭合的面积，环路面积与电感二者之间的关系可以用下图所示的右手法则和磁场来说明。   
![右手法则](http://picbed.moongazer.cn/hub/20260417112431040.png)
（图片来源：Hank Zumbahlen. staying-well-grounded）   

环路之内，沿着环路所有部分流动的电流所产生的磁场相互增强。环路之外，不同部分所产生的磁场相互削弱。因此，磁场原则上被限制在环路以内。环路越大则电感越大，将参考平面置于差分对下方是减小回路面积和阻抗的最佳方法。   

下图则显示了大多数交流电流在接地层中选取的路径，它所围成的面积最小，位于U形顶部走线正下方。实际应用中，接地层电阻会导致低中频电流流向直接返回路径与顶部导线正下方之间的某处。不过，即使频率低至1 MHz或2 MHz，返回路径也是接近顶部走线的下方。
![](http://picbed.moongazer.cn/hub/20260417112904457.png)
（图片来源：Hank Zumbahlen. staying-well-grounded）   


**破坏“最小电感路径的后果”**

如果走线正下方的地平面有一个裂缝或缺口，回流电流无法继续贴着走线正下方走，被迫绕过这个缺口。这个被迫的绕道会瞬间急剧增加电流的环路面积，导致局部电感突增。增大的电感不仅会引起阻抗不连续（导致信号失真和时序抖动），还会使这个巨大的电流环变成一个发射天线，向外辐射严重的电磁干扰（EMI）。

::: warning ❓ 那我问你：如果地平面被破坏，它“还会去哪”？
面试官冷笑一声：“回流电流总得闭环。你把它正下方的路挖断了，它会不会自己消失？”

::: details 💡 点击查看满分回答
**它不会消失，只会“绕路闭环”，而且通常以更糟糕的方式闭环。**

1. **沿参考平面边缘绕行**：
当差分线正下方参考平面被切断时，回流电流会沿裂缝边缘寻找可导通路径，绕到最近的连续铜皮区域再返回源端。路径一旦变长，环路面积立刻增大，等效电感上升，局部阻抗突变加剧。
2. **跨平面电容耦合“跳过去”**：
如果上下相邻平面之间存在分布电容，特别是电源平面与地平面之间有足够去耦网络，部分高频回流会通过位移电流跨层耦合完成闭环。但这条路径对频率和去耦布局高度敏感，稳定性远不如连续参考平面。
3. **耦合到邻近导体形成寄生回路**：
在严重不连续场景下，回流还可能借道邻近信号线、屏蔽层或连接器外壳等寄生路径。此时共模电流显著增加，表现为辐射变强、串扰上升、眼图恶化，且问题往往“偶发且难复现”。

**总结**：地平面被破坏后，回流电流并不会“没地方去”，而是被迫走阻抗更高、环路更大的替代路径。工程后果通常是阻抗不连续、时序裕量下降和 EMI 恶化同时出现。最优解不是“事后补救”，而是在布局阶段保证参考平面连续，并在必要跨分割处提前设计回流桥接与就近去耦。

:::

## 7. 总结

在原理图中，导线仅仅是一条连接两点的完美直线，但在高频数字信号设计的现实世界里，它有了一个响当当的名字——传输线。在《Signal and Power Integrity  Simplified》这本书中，通篇的内容都在告诉我们：从低频跨入 MHz 甚至 GHz 的高速频段时，再不能仅靠电分中的基尔霍夫定律分析客观问题了，麦克斯韦的电磁场理论将接管了整个 PCB Layout 的统治权。

回望差分信号的原理，本质就是一场极其精妙的电磁学交易。它用翻倍的引脚数量和极其严苛的布线规则（等长、等距、同层），换取了对电磁能量的绝对控制权。
