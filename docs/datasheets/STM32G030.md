# 芯片设计记录：STM32G030F6P6

::: info 文档信息
*   **芯片型号**: STM32G030F6P6
*   **制造商**: STMicroelectronics
*   **Datasheet 版本**: Rev 3, 2021
*   **一句话简介**: 主流 ARM Cortex-M0+ MCU，带 32KB Flash，TSSOP20 封装
:::

---

## 1. 核心电气参数 (Electrical Specs)

::: danger 极限参数警告
设计原理图和焊接前**必须核对极限参数**，防止过压或过流烧毁芯片。
:::

*   **供电电压 (VDD/VCC)**
    *   推荐工作电压：<span class="hw-tag">2.0V ~ 3.6V</span>
    *   绝对最大额定电压：`4.0V`
*   **引脚耐压 (Pin Tolerance)**
    *   是否兼容 5V？是 (部分标识为 FT 的引脚兼容)
*   **IO 电平阈值**
    *   $V_{IH}$ (输入高电平最小)：`0.7 * VDD`
    *   $V_{IL}$ (输入低电平最大)：`0.3 * VDD`
*   **功耗预估 (Current Consumption)**
    *   典型工作电流 (Active)：`3mA @ 64MHz`
    *   休眠电流 (Sleep/Standby)：`2.1uA`

## 2. 关键引脚分配 (Key Pinout & Config)

| 引脚名称 | 序号 | 类型 | 必须的外围配置 / 备注 |
| :--- | :---: | :--- | :--- |
| VDD | 1 | PWR | 必须紧靠引脚放置 `100nF` 的去耦电容 |
| VSS | 2 | PWR | 接地 |
| NRST | 3 | IN | 低电平复位，内部已经有 `40kΩ` 上拉电阻 |
| BOOT0 | 4 | IN | 启动模式选择：下拉 10k 为 Flash 启动，拉高为 ISP 启动 |
| I2C_SDA| 12 | OD | 如果用作 I2C 必须加上拉 `4.7K` |

## 3. 硬件配置参考

::: tip 硬件布线建议 (Layout Guidelines)
*   去耦电容务必放置在紧贴 VDD 引脚的地方。
*   晶振布线必须包地处理。
:::

## 4. 异常与避坑 (Gotchas)

::: warning 常见坑点
使用内部 ADC 时，配置采样周期至少需要设置在 `12.5` 个周期以上，否则如果在高速 I/O 跳变时容易导致采样值波动！
:::
